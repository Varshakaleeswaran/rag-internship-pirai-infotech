\# RAG Pipeline – Internship Notes



\## What is RAG?

Retrieval-Augmented Generation combines retrieval from external

knowledge sources with text generation.



\## Why RAG is Important?

\- Reduces hallucination

\- Enables use of private data

\- Improves answer accuracy



\## Core Components

1\. Document Loader

2\. Text Chunking

3\. Embedding Model

4\. Vector Database (ChromaDB)

5\. Retriever

6\. Large Language Model



\## RAG Flow

User Query → Embedding → Vector Search → Relevant Chunks → LLM → Answer
